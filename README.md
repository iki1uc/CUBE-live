# 1. Die 4 Eingaben

Jeder nanoWÜRFEL benötigt exakt vier Werte:

1. **Startwert**

`10, -5, +5, +1`

Der nanoWÜRFEL rechnet immer in derselben Reihenfolge:

### 1. IMPULS  
Startwert übernehmen.

### 2. WIDERSTAND  
Startwert + Minuswert.

### 3. GLEICHGEWICHT  
Ergebnis + Pluswert.

### 4. DURCHBRUCH  
Ergebnis + Schwellenwert.

### 5. WIRKUNG  
Finales Ergebnis.

Start = 10  
Minus = -5
